## 使用说明
本仓库提供了多个入门案例, 默认启动案例为 `Electron 渲染案例`

### Electron 启动案例
**案例说明**<br>
最小的可启动案例，用于熟悉 Electron 的程序入口，启动后将会在控制台看到打印信息

**快捷启动**<br>
`npm run start-hello`

**相关文件**<br>
`hello/`

### Electron 渲染案例
**案例说明**<br>
最小的可启动应用案例，启动后将会打开一个 Electron 应用程序

**启动方式**<br>
`npm run start-browser`

**相关文件**<br>
`browser/`

### Electron Preload 案例
**案例说明**<br>
最小的可启动 Preload 案例，用于熟悉 Electron 的 Preload 功能

**快捷启动**<br>
`npm run start-preload`

**相关文件**<br>
`preload/`
